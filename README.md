### Prerequisites

In order to run this project you need:
1. Code editor
2. Web browser
3. Git

### Setup

Clone this repository to your desired folder:
Run the below command:

```sh
cd <your disered folder name >
git clone https://github.com/the-faizmohammad/Capstone-API-based-webapp
```
